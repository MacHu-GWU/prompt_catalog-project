Install
------------------------------------------------------------------------------

``prompt_catalog`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install prompt-catalog

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade prompt-catalog
